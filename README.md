### 【美预】小程序前端

> 本项目是一个美业预约小程序uniapp项目

## 技术选型及其哲学

- 约定大于配置的设计理念（uni-helper-pages基于文件的路由、components, utils, stores目录组件自动导入）
- 代码格式化及提交统一风格（@antfu/eslint-config）
- env文件类型提示补全（vite-plugin-env-parse）
- 充满信心的书写代码（typescript）
- 超快的开发构建速度（vite）
- 简单可信赖的状态管理（pinia）
- 组合式思路写js（vue3-script setup）
- 组合式思路写css（unocss）
